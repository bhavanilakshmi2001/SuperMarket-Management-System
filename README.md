🛒 Supermarket Management System

📖 Overview

A web-based Supermarket Management System to manage products, generate invoices, and handle online product purchases efficiently.

---

⚙️ Tech Stack

- Java
- JSP & Servlets
- MySQL
- HTML, CSS, JavaScript
- Apache Tomcat

---

🚀 Features

- 🛍️ Product Management (Add, Update, Delete, View)
- 🧾 Invoice Generation
- 🛒 Online Product Purchase
- 📦 Manage product stock and availability

---

🏗️ Modules

- Product Module
- Billing / Invoice Module
- Purchase Module

---

▶️ How to Run

1. Import project into Eclipse / IntelliJ
2. Configure MySQL database
3. Deploy on Apache Tomcat
4. Run in browser

---

💡 Learning Outcome

- CRUD operations implementation
- Invoice generation logic
- Database integration using MySQL
- Real-time project development using JSP & Servlets
